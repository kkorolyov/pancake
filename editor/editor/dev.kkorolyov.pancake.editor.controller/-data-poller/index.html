DataPoller

class DataPoller : Controller
Content copied to clipboard

Regularly polls and maintains information from a registered GameLoop. Exposes polled information as observable values. Polls at rate set in platform Config.get as editor.pollRate - target number of polls per second, else defaults to 30 polls per second.
