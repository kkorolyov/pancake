Atlas

class Atlas @JvmOverloads constructor(buffers: Collection<PixelBuffer>, spacing: Int? = null) : AutoCloseable

A collection of PixelBuffers packed as a single buffer. Provides indexed viewports to packed buffers in the order the buffers were provided. Can provide custom spacing px to add between backed buffers - defaults to a value derived from the maximum buffer dimensions.

data class Viewport(val s0: Double, val t0: Double, val s1: Double, val t1: Double)

A viewport on those pixels in an Atlas pertaining to an individual buffer. Coordinates are in the interval [0.0, 1.0] where (0.0, 0.0) is the origin of the overall atlas, and (1.0, 1.0) is the corner diagonal from the origin.
